# 0.1.0 (2025-05-07)

### Features

* **InlineReasoningTrace:** Initial release of InlineReasoningTrace component
  - Displays AI reasoning steps in a clean, hierarchical format
  - Supports expandable/collapsible accordion behavior
  - Individual step expansion with "Show more/Show less" functionality
  - Configurable step visibility management
  - Processing state with FLIP and fade animation modes
  - Task type icons for both development and reasoning workflows
  - Governance indicators and warning overlays
  - Smooth animations and transitions
  - Full TypeScript support
