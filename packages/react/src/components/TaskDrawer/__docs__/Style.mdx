## Typography

All task labels and UI text are set in sentence case.

| Element | Type token | Notes |
| ------------------- | ----------- | -------------------------------------- |
| Plan title | `$label-01` | Single line, truncated with ellipsis |
| Preview | `$label-01` | Single line, truncated with ellipsis |
| Task row label | `$label-01` | Strikethrough for `complete` / `dropped` mutation |
| Task row index | `$label-01` | Fixed-width, right-aligned column |
| Mutation tag | `$label-01` | — |
| "Review" action | `$label-01` | — |
| Skip link | `$label-01` | Visually hidden until `:focus-visible` |

### Body

The body's maximum visible height is set via CSS custom properties:

| Custom property | Value | Notes |
| ---------------------- | ---------- | ---------------------------------- |
| `--ptl-row-height` | ≈ 33 px | `$spacing-05` + 2 × `$spacing-03` + 1 px divider |
| `--ptl-max-body-height` | ≈ 148 px | 4.5 × `--ptl-row-height` — 4.5 rows visible before scrolling |

## Motion

| Animation | Duration | Easing | Trigger |
| --------------------------- | -------------------------------- | --------------------------------- | ------------------------------ |
| Drawer mount slide-up | `$duration-moderate-02` (240 ms) | `motion(standard, expressive)` | Component first mounts |
| Body expand / collapse | `$duration-moderate-02` (240 ms) | `motion(standard, expressive)` | Toggle |
| Preview animate in / out | `$duration-moderate-02` (240 ms) | `motion(standard, expressive)` | Toggle |
| Status icon cross-fade | `$duration-fast-02` | `motion(standard, productive)` | `planStatus` change |
| Spin (generating / updating) | 1 000 ms linear infinite | — | While status is active |
| Task row fade-in | `$duration-moderate-02` (240 ms) | `motion(standard, expressive)` | First time row enters viewport |
| Status icon dissolve-out | `$duration-fast-02` (120 ms) | `motion(standard, productive)` | Status change (exit phase) |
| Status icon dissolve-in | `$duration-moderate-02` (240 ms) | `motion(standard, expressive)` | Status change (enter phase) |
| Mutation tag scale-in | 150 ms | `motion(standard, expressive)` | Tag added |
| Skeleton shimmer | 3 000 ms linear infinite | — | While `generating` |
